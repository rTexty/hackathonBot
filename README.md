## Installation

1. Способ 

Установка через GitHub

```bash
git clone https://github.com/rTexty/hackathonBot
```
```bash
python3.12 -m venv venv
source venv/bin/activate
pip install requirements.txt
```
Заходите в main.py
```bash
python3.12 main
```


2. Способ
Развернуть через DockerHub
```bash
docker pull rtexty/surgpu
```
Запуск 

```bash
docker start surgpu
```
